Quanda

I built a program that makes learning easier with automatic flashcard generation from any article link using ML.

Overview

I know a lot about learning and a lot about machine learning. So I combined those 2 and made somethng that'll help people learn more effectively.

The Python script has the user enter a link to a Wikipedia or Medium article and generates an Anki flashcard deck using NLP. The deck is made up of fill-in-the-blank questions and answers which the user can add to their Anki app by clicking the downloaded file.

My next steps for this project will be to make a fully functional web app and add support for PDF files. The goal is to enable motivated students to easily learn anything. Imagine getting flashcards made for your entire science textbook within 30 seconds and just reviewing that for the whole semester. 🤯
